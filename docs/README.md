These docs describe Agent Commons as the broader system for human, AI, and collective coordination across scales:

- [project-vision.md](./project-vision.md)
- [relational-agency-and-holons.md](./foundations/relational-agency-and-holons.md)

## 2. Project Governance Layer

These docs describe the currently implemented governance/documentation layer inside Agent Commons:

- [agent-commons-meta-protocol.md](./governance/agent-commons-meta-protocol.md)
- [project-bootstrap-spec.md](./governance/project-bootstrap-spec.md)

## How To Read This Repo

- Start with the umbrella layer if you want to understand what Agent Commons is trying to enable.
- Start with the governance layer if you want to use the current tooling in another project.
- Treat the governance layer as one implemented slice of Agent Commons, not the whole system.
