# ADR-0049 — Reproduction-continuity Primitive Admission

## Status

active (authored + activated 2026-04-22 under `canon-rebuild-phase-3b-4` decision-gated plan; Option B — new primitive — per operator selection post decision-brief)

## Context

Phase 3b.1 (Care, ADR-0045 Option F), Phase 3b.2 (Norms, ADR-0046 Option B), Phase 3b.3 (Power, ADR-0047 Option G.full), Phase 3b.3b (Johar integration, ADR-0048) closed same-day 2026-04-22. Phase 3b.4 addresses the last capstone §3 Tier-1/2 disposition gap: reproduction-continuity.

### Capstone evidence

Current capstone (`docs/research/corpus-review/research-capstone.md`, HEAD `5bd00c5` 2026-04-22) classifies reproduction-continuity as a primitive-admission candidate. Three capstone sections converge:

- **§1 Convergences** (line 30): *"Precariousness / viability / ongoingness is the tacit substrate of 'coordination' talk. ... Current canon is silent on reproduction/continuity. The life-sciences-adjacent convergence argues a coordination primitive set that omits reproduction/continuity cannot account for what it coordinates for — strongest single argument for admitting `reproduction-continuity`, continuous with ADR-0008's field substrate."*

- **§3 Reproduction/continuity** (lines 108-113):
  - Tradition-count: *"the* central primitive in Care Ethics / Feminist Economics (Federici 'ground zero of revolution'; Folbre); load-bearing in commoning-Commons (De Angelis), Viable System Model ('viability *is* survival over time'), Pluriversal (cyclical rituals, ancestor rebirth), Governance-Process (succession), Structured Disagreement (replication crisis)."*
  - Structural role: *"invisible-substrate (Care, commoning); viability condition (VSM); renewal regime (Governance-Process, Structured Disagreement); agent-property via adaptivity in Autopoiesis-adjacent work."*
  - Paraphrase test: **partial**. *"'Repeated commitments over generations + knowledge-transmission' captures pattern but misses Federici's point that reproductive labor is **invisible** to the accounting frame."*
  - **Verdict**: **admit as primitive**. *"The corpus support is strong enough that treating this only as a field-property understates the convergence."*

- **§7 Frozen-concepts v3 proposal** (line 245 heading "Admit as primitive (strong cross-tradition support)", line 251): `reproduction-continuity` *"differentiated from `reproductive-commoning` because the frozen slug names the care/commoning lineage specifically, while this candidate names the broader cross-tradition primitive of renewal, survival-over-time, and replication."*

- **§8 Tier 1 item 5** (line 315): *"Reproduction / continuity primitive. ... without a named continuity primitive, Spore can describe coordination episodes but not the renewal, succession, and replication conditions they are for."*

- **Appendix Fix 2 "Reproduction/continuity reconciliation"** (lines 378-381) records the explicit v1→v2 position change: from v1's field-only framing to v2's primitive-admission. This invalidates the plan's original field-property-default framing; the decision-brief corrected for this.

### The parsimony question

Per ADR-0048 (Phase 3b.3b), parsimony at N primitives is an earning-test outcome, not an axiom. Phase 3b.1/3b.2/3b.3/3b.3b established precedent for softening capstone primitive-verdicts when the earning test judged candidates marginal on (a) protocol-surface-specifiability. Phase 3b.4 applied the earning test honestly:

- **(a) Protocol surface specifiability**: **marginal**. Candidate operations — enact-succession (BKC stewardship transfer; Debian DPL-succession); transmit-knowledge (apprenticeship; doc DAGs outliving authors); replicate-commitment-pattern (monthly rituals; annual review cycles); regenerate-field-capacity (reproductive labor sustaining coordination substrate) — are thinner than Intent's `propose`, Commitment's `bind`, Evidence's `attest`, Signal's `broadcast`. But Federici's specific point (*Caliban and the Witch*, 2004; "Wages Against Housework," 1975) is that this very thinness IS the phenomenon: reproductive labor appearing operation-less is how accounting-frames invisibilize it. The canon-legible countermove is to name reproduction as first-class coordination verb — the substitution-trap discipline admitted in ADR-0048 catches this kind of silent-delegation-to-invisibility.

- **(b) Multi-scale operational implementations across Spore's instance families**: **strong**. BKC/Octo: herring-cycle stewardship rotation; Landscape Hub facilitator-role transfer; KOI federation peer-re-signing. Poietic Match: commitment-pool renewal cycles; matcher-persona lifecycle; user-retention renewal mechanics. Personal-operator/DW: doc DAG renewal; session-to-knowledge-episodes transition; calendar-rhythm (morning/evening brief); vault mentionedIn resync. Federation-level: protocol-version compatibility, credential rotation, membership renewal. Four scales; distinct reproduction regimes at each; (b) passes more strongly than any prior 3b earning-test.

**Net**: marginal on (a) acknowledged honestly; strong on (b). Total judgment under 3b.3b discipline: admission is earned, with (a) marginality named as a canon-transparent property. The alternative options (field-property per Option A, mode-across-primitives per Option D, decline per Option G) were evaluated in decision-brief §Option A vs Option B vs Option D; Option B chosen because Option D risks reproducing the invisibilization pattern by dissolving reproductive work into continuity-properties-of-other-things, while Option A takes the field-property framing capstone v2 explicitly rejected.

### Relationship to ADR-0002 (reproductive-commoning doctrine lens)

ADR-0002 (2026-04-18) promoted `reproductive-commoning` from v2-vocabulary entry to cross-cutting doctrine lens — a practice-lens for making-reproductive-labor-visible across existing primitives. ADR-0049 is explicitly **extensional, not supersessionary**: the doctrine lens remains authoritative at practice-layer; ADR-0049 adds the primitive-layer that the lens lenses over. The two coexist:

- **Reproductive-commoning (ADR-0002)**: cross-cutting doctrine. The practice-lens of making reproductive labor visible across existing coordination primitives (field-as-reproductive-apparatus; relational-agency as commons-as-verb; federation as commoning-mechanism).
- **Reproduction-continuity (ADR-0049)**: 8th primitive. The coordination verb reproductive labor IS — making the substrate of the verb-loop persist across time, actor turnover, and generational succession.

Phase 3b.4 consequences: any future Spore canon that names reproductive-labor phenomena (e.g., BKC stewardship rotation, Poietic Match pool renewal, DW vault mentionedIn resync) now has a primitive-level vocabulary for the labor itself, in addition to the doctrine-lens for its visibility.

### Relationship to ADR-0045 (care-commoning) and the care-reproduction gap-residue

Capstone §3 line 113 explicitly reserves the care-reproduction distinction as residue: *"The care/reproduction distinction remains a residue."* Phase 3b.4 must articulate this per capstone-required caveat (AC12 of the plan).

The distinction stated cleanly:

- **Care-commoning (ADR-0045, cross-cutting doctrine)**: lenses the **asymmetric-relational** axis — who attends to whom, who bears the vulnerability-absorption cost, who receives care. Tronto's four-phase ethic (attentiveness / responsibility / competence / responsiveness). Asymmetric by construction (not symmetric reciprocation).

- **Reproductive-commoning (ADR-0002, cross-cutting doctrine)**: lenses the **renewal-over-time visibility** axis — what labor sustains the coordination substrate across generations, and how that labor is kept (or made) visible in the accounting frame. Federici / Folbre / Kittay / Bresnihan lineage. The political economy of care-labor-plus-provisioning-labor-plus-maintenance-labor considered as the substrate of the commons.

- **Reproduction-continuity (ADR-0049, 8th primitive)**: names the **coordination verb** the two doctrines lens over. The first-class coordination labor of sustaining-across-time. Beer/VSM viability-over-time + Pluriversal cyclical-ritual + Governance-Process succession-rules + Structured-Disagreement replication.

The three elements overlap at **reproductive-care-work** — the specific intersection Federici names as most invisibilized and most power-captured — but none reduces to another:

- Care-commoning operates on care-relations-that-are-not-reproductive (e.g., crisis-response attending to acute vulnerability; non-substrate-sustaining care).
- Reproductive-commoning operates on reproductive-labor-that-is-not-care-labor (e.g., infrastructure-renewal; technical-credential-rotation; knowledge-archive-transmission where no caregiver-dependent asymmetry is present).
- Reproduction-continuity operates at verb-layer whether or not visibility-practices are active (the verb names the labor; the doctrines name practices that make its visibility non-optional).

This gap-residue is canon-legible going forward: any canon edit touching one of the three must preserve the distinction by prose clarity. ADR-0049 does not resolve the residue — it provides the structural vocabulary within which the residue can be named cleanly. Future canon phases (Phase 3b.7 glossary refinements bundle; sensor-and-oracle-governance foundation doc in Phase 4) should preserve this three-way distinction.

### Relationship to ADR-0047 (Power multi-layer decomposition) and Federicist power-over-reproduction theme

ADR-0047 admitted `asymmetric-commitment` as a power-asymmetry axis on Commitment-party-structure (Layer 2), citing Baier / Folbre / Kittay as the asymmetric-vulnerability lineage — a specifically Federicist critique. Prior to ADR-0049, the reproductive-labor substrate the asymmetry operates over was implicit: asymmetric-commitment-where-reproductive-labor-is-disproportionately-borne was a named pattern but the labor itself was not a first-class primitive.

ADR-0049 makes the substrate explicit. Power-over-reproduction — the specifically Federicist power-theme — now operates through `asymmetric-commitment` **over** the newly-admitted reproduction primitive. ADR-0047 Layer 2 (asymmetric-commitment) now has explicit primitive-layer substrate (reproduction) at the labor-locus where Federici / Folbre / Kittay locate the primary power-capture. This strengthens ADR-0047's Federicist-integration without modifying ADR-0047's structure.

### Relationship to ADR-0046 (Field rule-level stratification)

ADR-0046 stratified Field into Ostrom's three rule-levels (operational / collective-choice / constitutional). Reproduction-as-primitive interacts: constitutional rules typically have high reproduction-labor (slow-change; generational succession; ancestor-authority-style); operational rules have low reproduction-labor at the level (high-change moment-to-moment) but ongoing reproduction at the rule-enactment layer (each operational-rule-enactment IS reproduction of the rule-in-use vs. rule-in-form). Reproduction-continuity and rule-level-stratification compose: reproduction of rule-in-use is distinct from reproduction of rule-in-form, and the distinction clarifies where reproductive labor operates.

### Paraphrase test — explicit result

Capstone §3 paraphrase: *"Repeated commitments over generations + knowledge-transmission captures pattern but misses Federici's point that reproductive labor is invisible to the accounting frame."*

**ADR-0049 paraphrase-test verdict**: **captured**. Admitting reproduction-continuity as first-class primitive is precisely the canon-legible countermove to invisibility-in-the-accounting-frame. The paraphrase-test's partial-failure under prior canon (repeated-commitments-over-generations + knowledge-transmission-via-evidence) now reads as a *protocol-surface summary* of reproduction's operations at the (a) side of the earning test, with the Federicist point directly addressed by the primitive admission itself. Federici's invisibility critique does not survive primitive-admission because the admission names the labor; the reader of the grammar now has a first-class vocabulary for the labor Federici named invisible.

## Decision

**edit.** Spore canon admits `reproduction-continuity` as the 8th primitive — a fifth coordination verb operating at a distinct time-scale from the within-episode verb-loop. Canon structure becomes: 3 structural primitives (field, holon, membrane) + 5 coordination verbs (intent, commitment, evidence, signal, reproduction) = 8 primitives.

**Concept slug admitted**: `reproduction-continuity` (capstone's preferred candidate, differentiated from frozen `reproductive-commoning` which names the doctrine-lens). `viability-over-time` is the Beer-lineage synonym noted in glossary; not a separate canon slug.

**yaml vocabulary bump**: `docs/research/concepts-p2p-wiki.yaml` v6 → v7.

## Consequences

- **Spore canon moves 7 → 8 primitives for the first time since ADR-0044** (core thesis primitive-roster alignment, 2026-04-22). This is the first exercise of the "roster extends when earning-test passes" opening that ADR-0044 + ADR-0048 jointly established. The move is documented as an earning-test outcome, marginal on (a) with acknowledged Federicist-counterpoint that names the marginality as the phenomenon.

- **Care-reproduction gap-residue addressed** per capstone §3 line 113 caveat (AC12). Three distinct canon elements now carry the three-way distinction: care-commoning (ADR-0045 doctrine lens on asymmetric-relational axis), reproductive-commoning (ADR-0002 doctrine lens on renewal-over-time-visibility axis), reproduction-continuity (ADR-0049 primitive at coordination-verb layer). Reproductive-care-work is the overlap that Federici locates; the three-way distinction preserves it without collapsing. Canon going forward: any text touching reproduction / care / commoning distinguishes which layer (verb, practice-lens, or asymmetric-relation lens) is active.

- **Power-over-reproduction Federicist theme** (ADR-0047 Layer 2 intersection): `asymmetric-commitment` now has explicit primitive-layer substrate at the reproduction-labor locus. The Federicist critique (reproductive labor as primary capture site) now has canon-legible structural machinery: ADR-0047 names the power-asymmetry axis; ADR-0049 names the labor-substrate the asymmetry operates over; ADR-0002 names the visibility-practice that counters the capture; ADR-0045 names the relational-practice adjacent to but distinct from the reproductive.

- **ADR-0002 preserved as doctrine-lens** (extended, not superseded). The practice-lens of making reproductive labor visible remains authoritative at doctrine-layer. ADR-0049 provides the structural vocabulary the lens operates over. Both coexist. The combination is canon-richer than either alone: the primitive names the labor; the doctrine names the practice of making it accountable.

- **Paraphrase-test partial-failure** under pre-ADR-0049 canon is now **captured**: the capstone §3 gap (reproductive-labor-invisible-to-accounting-frame) closes because the accounting-frame (the primitive grammar itself) now has a first-class vocabulary for reproductive labor. Federici's specific capstone-cited critique — that paraphrasing reproduction into "repeated commitments + knowledge-transmission" preserves the invisibility — is answered by the primitive admission itself.

- **Earning-test (a) marginality** is preserved as canon-transparent property. Reproduction-continuity's protocol-surface is genuinely thinner than Intent/Commitment/Evidence/Signal. The thinness is not a defect of the ADR; it is the property that required the Federicist-counterpoint to earn admission. Future glossary work should name the operations (enact-succession / transmit-knowledge / replicate-commitment-pattern / regenerate-field-capacity) as reproduction-verb-sub-operations without over-specifying them into separate primitives (which would replicate the problem ADR-0048 solved with modes-across-primitives).

- **Downstream canon references to "seven primitives" or "intent, commitment, evidence, signal"** have been updated in the two primary canon surfaces (project-vision.md Core Thesis + constitutional-artifacts coordination-ecology) and their cross-references. Other canon surfaces (foundations/lexicon/field.md, holonic-network-architecture.md, mycorrhizal-federation-protocol.md retained-as-reference-to-federation-protocol-per-ADR-0043, relational-agency-and-holons.md) preserve their prior framings; they reference primitives by name, not by count, and remain correct. A follow-up propagation sweep is Phase-3b-parking-lot level work, not Phase 3b.4 scope.

- **IC / PM coordinated-update of their grammars**: deferred per plan Non-goals. IC and PM reference Spore's primitive grammar but have not ratified the 7→8 move; Phase 3b.5+ may address coordinated propagation if warranted. The 7-primitive-roster ADR-0044 cross-references (IC:ADR-0044, PM:ADR-0044) are not auto-superseded by this Spore-local move; each project decides independently whether to follow.

- **Phase 3b.5-3b.8 and Phase 4 implications**: Identity-as-holon-property (3b.5) may reference reproduction (continuous-identity-across-reproduction is a named VSM S5 + Autopoiesis concern); Reciprocity + Trust (3b.6) may reference reproduction-based reputation-continuity; Glossary refinements (3b.7) should fine-tune the reproduction-verb sub-operations. Johar rewilding thesis (3b.8) has reproduction-at-field-scale interaction. Phase 4 sensor-and-oracle-governance foundation doc should name the reproductive-Evidence reading (longitudinal attestation; replication-regime) as explicit subspecies of Evidence operating in reproduction-continuity context.

- **Encounter-as-8th-primitive dedicated ADR** candidate (parked from 3b.3b earning-test): Encounter's admission-question now recontextualizes as 9th-primitive admission-question, not 8th. This is information for that future plan; no action in 3b.4.

- **Validator expectation**: 9/30 baseline held. Edits are content-only on canon files already in-scope; no new doc_id admissions.

- **Instance-family operational implications**: BKC/Octo bioregional stewardship now has reproduction-verb vocabulary for herring-cycle rotation, facilitator-succession, and protocol-version-renewal; Poietic Match has reproduction-verb vocabulary for pool renewal, matcher-persona-lifecycle, user-retention-as-reproduction; personal-operator has reproduction-verb vocabulary for doc DAG renewal, session-to-persistent-knowledge, vault-resync. Canon upgrade enables clearer surfacing of reproduction-labor at each instance; earning-test (b) evidence is what the vocabulary now covers.
