# ADR-0052 — Reciprocity + Trust as Derived Glossary Slugs with Shared Framing-Note (Option C)

## Status

draft (authored 2026-04-22 under `canon-rebuild-phase-3b-6` decision-gated plan; Option C — both-as-glossary + shared framing-note with residue-flagging per operator selection post decision-brief)

## Context

Phase 3b.6 in canon-rebuild Phase-3b sequence. Bundled disposition for capstone §3 Tier-3 derived-level items Reciprocity (line 106) and Trust (line 57). Both capstone verdicts are single-limb and non-controversial:

- **Reciprocity §3 verdict** (verbatim): *"admit as derived (paired commitments); flag residues."*
- **Trust §3 verdict** (verbatim): *"admit as derived with glossary entry flagging residue."*

Both admit as derived; both mandate explicit residue-flagging. The canonical question bundled in ADR-0052 is *how* that derived-admission is represented at canon-layer given the evolving Phase-3b-1-through-3b-5 machinery under which "paired commitments" (Reciprocity's capstone-form) and "accumulated evidence/signals → commitment-willingness" (Trust's capstone-form) now compose.

### Post-3b.1-5a reciprocity composition

The capstone was authored before Phase 3b landed. At authorship time, "paired commitments" meant single-mode symmetric-dyadic commitments between individuals. In current canon (HEAD `e2751d7`, post-ADR-0051), a reciprocity-pair can be one of three distinct compositions:

1. **Symmetric-dyadic** — two Commitment primitives, both individually-authored, with symmetric obligation (REA/ValueFlows duality; Axelrod Tit-for-Tat; bilateral-exactly-once in Distributed Systems).
2. **Asymmetric-commitment-pair** — two Commitment primitives with structurally-asymmetric obligation running in both directions at unequal weight (ADR-0047 Layer 2 asymmetric-commitment machinery; captures caregiver→cared-for with cared-for→caregiver counter-obligation; Kittay dependent-carer with nested dependencies).
3. **Joint-commitment-at-both-ends** — two joint-commitments, each authored at the joint/collective-compositional layer (ADR-0050; federation-to-federation protocol-version-adoption is the cleanest operational case — each federation's agreement is itself multi-party-simultaneous).

This three-mode composition is the load-bearing reason Option C was selected. The derived-admission cannot collapse into a single-mode definition without losing structure that post-3b.1-5a canon already carries.

### Trust's post-3b.5 interaction

Trust-of-whom is a second-order question now that identity is admitted as a relationally-constituted Holon whole-emergent-property (ADR-0051). Trust is a relational-stance between Holons whose identity is itself relational; this does not require elevating Trust beyond derived-admission but does require the framing-note to name the second-order dependency so bridge-note authors referencing `trust` downstream can route correctly.

### Tradition-coverage summary

**Reciprocity** — 8 load-bearing traditions: Commons Governance (Ostrom 1998); Pluriversal (*ayni*, *kintu*, "*the* core ritual primitive"); Trust Networks (Hardin "mutually reinforcing"); Care Ethics (Noddings non-contractual); REA/ValueFlows (duality axiom); ABM (Axelrod Tit-for-Tat); Distributed Systems (bilateral/exactly-once); Autopoiesis/4E (coordination-of-coordinations). Weak/absent: Philosophy of Collective Agency (implicit); MARL (only emergent); VSM ("absent as named"); Category Theory; Canonical Layering; Governance-Process ("mostly absent — CLAs one-directional"); Repo-Topology; Audience-Scoping; Structured Disagreement. Paraphrase-test: *adequate structurally*.

**Trust** — 4 load-bearing (Trust Networks Luhmann/Hardin/Uslaner; Commons Governance Ostrom 1998 triad; Care Ethics / Feminist Economics Kittay; Governance-Process Apache "Earned Authority"); 2 peripheral (Philosophy of Collective Agency Baier-critiqued absence; VSM late-import); 1 explicit negation (Distributed Systems "trustlessness"); 10 absent. Paraphrase-test: *partial* — commitment-willingness-given-evidence/signals captures rational-expectation + engineering-displacement; loses Luhmann-uncertainty + Baier-asymmetric-vulnerability.

## Consequences

### ADR lineage (related_adrs)

- **ADR-0001 (pluriversal-incommensurability):** cross-referenced — R-Rec-1 (Pluriversal ritual-with-non-human-kin reciprocity) sits within ADR-0001's ontological-pluriversality held-tension cluster. 3b.6 does NOT resolve this tension; Pluriversal-ritual-reciprocity remains marked-as-irreducible, partially captured via care-commoning + derived-reciprocity + asymmetric-commitment composition, with residue explicit. Same dual-operability pattern ADR-0050 and ADR-0051 established for pluriversal-joint-commitment and pluriversal-kinship-identity respectively.
- **ADR-0044 (core-thesis-primitive-roster):** unchanged — 9-primitive roster (3 structural: field / holon / membrane + 6 verbs: intent / commitment / joint-commitment / evidence / signal / reproduction) preserved. No primitive admission; no doctrine admission; no mode admission; no property admission. Canon-body untouched.
- **ADR-0045 (care-commoning doctrine):** strengthened via residue-capture — R-Rec-2 (Noddings non-contractual mutuality) + R-Tr-2 (Baier/Kittay asymmetric-vulnerability) are within care-commoning's doctrine-lens scope. Framing-note makes explicit that care-commoning (doctrine-lens) captures the relational-ethical dimension while derived-Reciprocity + derived-Trust capture the paired-structure + commitment-willingness structure respectively; neither reduces to the other. Three-way articulation: care-commoning lens + reciprocity-derived + trust-derived, each naming-what-it-is without collapse.
- **ADR-0047 (power-multi-layer-decomposition):** composed-over — Layer 2 asymmetric-commitment is the operational machinery for the asymmetric mode of the three-mode reciprocity composition. Framing-note maps asymmetric-reciprocity = asymmetric-commitment-pair + counter-obligation direction. R-Tr-2 (Baier/Kittay) intersects Layer 2 at the trust-side of asymmetric-commitment relations.
- **ADR-0049 (reproduction-continuity primitive):** orthogonal — no new intersection introduced. Care-reproduction gap-residue three-way articulation (reproductive-commoning / care-commoning / reproduction-continuity per ADR-0049 §Consequences) extended conceptually: care-commoning now captures reciprocity-residue R-Rec-2 + trust-residue R-Tr-2 as well, but this is acknowledgment without re-disposition.
- **ADR-0050 (sociality-side-b-plus-primitive):** composed-over — joint-commitment is the third mode of the three-mode reciprocity composition. Reciprocity's paired-commitments-admission is compatible with joint-commitment at either or both ends; the pair-structure is preserved but elements may be joint rather than individual.
- **ADR-0051 (relational-identity-holon-property):** second-order interaction — Trust-of-whom is trust of a relationally-constituted Holon. Framing-note names this dependency without introducing new canon machinery; no elevation of Trust beyond derived required.

### Paraphrase-test verdicts (per capstone)

- **Reciprocity**: *adequate structurally* per capstone — paired-commitments-with-linked-evidence captures the symmetric REA/Axelrod reading. Three-mode post-3b.1-5a composition extends this adequacy to cover asymmetric-commitment-pair (ADR-0047 Layer 2) and joint-commitment-at-both-ends (ADR-0050) WITHOUT new primitive/property admission. Residues R-Rec-1 + R-Rec-2 flagged; capstone-mandate honored.
- **Trust**: *partial* per capstone — commitment-willingness-given-evidence/signals captures rational-expectation (Hardin) + engineering-displacement (Distributed Systems) + Apache earned-authority. Residues R-Tr-1 (Luhmann uncertainty) + R-Tr-2 (Baier asymmetric-vulnerability) flagged; capstone-mandate honored.

Partial-paraphrase-pass is correct shape for derived-terms flagging residue, analogous to ADR-0046's partial-paraphrase-pass for constitutive-structure Norms.

### Residue-flagging (MANDATORY per capstone — 4 named)

- **R-Rec-1 — Pluriversal ritual-with-non-human-kin reciprocity.** *ayni* offering to mountain-as-ancestor; *kintu* to land-beings. One "party" is not a signatory in Spore's commitment semantics (mountain does not sign). Partial capture via ADR-0001 held-tension cluster + ADR-0045 care-commoning extension to more-than-human care-relations (per Plumwood, De la Cadena). Residue explicit; unresolved.
- **R-Rec-2 — Noddings/Kittay care non-contractual mutuality.** Caregiver/cared-for reciprocity without contracting; reciprocity-asymmetric without formalized obligation. Partial capture via ADR-0045 care-commoning doctrine-lens + ADR-0047 Layer 2 asymmetric-commitment machinery; composition honored in framing-note.
- **R-Tr-1 — Luhmann trust-as-choice-under-irreducible-uncertainty.** Trust as functional-substitute for unattainable information; not reducible to accumulated-evidence because the whole point is acting *without* sufficient evidence. No canon machinery captures this directly; residue stands as irreducible-uncertainty flag. Named and explicit.
- **R-Tr-2 — Baier asymmetric-vulnerability.** "Accepted vulnerability to another's possible ill will or incompetence"; Kittay extension to dependent-care contexts. Partial capture via ADR-0045 care-commoning + ADR-0047 Layer 2 asymmetric-commitment. Relational-stance dimension overlaps with ADR-0051 relational-identity.

ADR-0052 Option C does NOT touch any of these. Commitment primitive bullet specifically preserved untouched so Phase 3b.7's three-senses work has clean ground to operate on. AC12 honored.
